## Квізи

### 1. Як працює наш розум? (Когнітивна психологія)
- Ефект Даннінга-Крюгера, когнітивний дисонанс
- Робоча пам'ять, ефект ореолу, ефект свідка
- Піраміда Маслоу, зона найближчого розвитку
- **Рівень:** Medium

### 2. Емоції та почуття
- Базові емоції за Екманом
- Емоційний інтелект (EQ)
- Механізми захисту (Фрейд)
- Стрес, вигорання, резилієнтність
- **Рівень:** Easy

### 3. Психологія особистості
- Типи темпераменту
- Big Five (OCEAN)
- Інтроверсія vs екстраверсія (Юнг)
- Локус контролю, self-efficacy
- **Рівень:** Medium

### 4. Соціальна психологія
- Конформізм (експеримент Аша)
- Підкорення авторитету (експеримент Мілгрема)
- Стенфордський тюремний експеримент
- Групове мислення, соціальна фасилітація
- **Рівень:** Hard

## Збереження результатів (localStorage)

### Структура даних
```json
{
  "quizResults": [
    {
      "quizId": "cognitive",
      "score": 8,
      "total": 10,
      "percentage": 80,
      "date": "2026-04-01T14:30:00",
      "timeSpent": 245
    }
  ],
  "settings": {
    "theme": "light",
    "name": "User"
  }
}
```

### Що зберігається
- Результати кожного проходження (квіз, бал, дата, час)
- Найкращий результат для кожного квізу
- Загальна кількість пройдених квізів
- Налаштування теми
